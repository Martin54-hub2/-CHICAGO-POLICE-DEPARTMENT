### ✅ Role-Based Access

**Admin:**
- Post department updates
- Add/edit/delete officers
- Manage callsigns & ranks
- Add infractions
- Full system access

**Supervisor:**
- Manage callsigns & ranks
- Add infractions
- Search officers
- Cannot post updates
- Cannot add/delete officers

**Officer:**
- File reports
- View personal dashboard
- See own infractions
- Read updates

- Search by name, username, callsign
- Click to view full profile
- Edit inline from results
- Real-time filtering

## 💡 USAGE EXAMPLES

### Add Officer
1. Admin Panel → "+ Add Officer"
2. Fill: username, password, name, callsign, rank, role
3. Officer can now login

### Edit Callsign
1. Search officer
2. Click "Edit" button
3. Update callsign field
4. Save → Updates everywhere

### Add Infraction
1. Search officer
2. Click "+ Infraction"
3. Enter reason and notes
4. Save → Shows on profile

### Post Update
1. Admin Panel → "Post Department Update"
2. Enter title and content
3. Post → Shows on all dashboards
